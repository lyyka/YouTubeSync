Your wireless solution for wacthing Youtube videos!

YoutubeSync allows you to watch youtube video synchronized accross unlimited number of devices connected on same network!

How It Works


A user creates a room and gets 7-digit room key.


Then, he shares the key and users connect to a room.



Now all of the users in the room are synchronized and are able to use all functions provided in the room.

How To Use

If you are a developer and want to start working on this project, here are a few notes to get you started!


Back-end

For back-end part, project utilizes Node.js , all of the server code can be found in /server.js file in root directory. Express framework for Node.js is used.
Default port for the server is 8000. You can change that in /server.js

Connection and synchronization

After setting up the back-end, our project must be able to somehow communicate between devices on same network.
To accomplish that, project uses socket.io library. It has all the awesome features needed to communicate between user and server including the feature to craete and manage rooms! That's exactly what we need.

Running the app

To run this app, all you need to do is open CMD, navigate to the apps root directory and run npm start. This command will start up the server.
Now, just go into your browser and type in [Your IPV4 address]:8000. And that's it, your app should be up and running! Remember, every time you make a change to /server.js you must restart the server from CMD.
